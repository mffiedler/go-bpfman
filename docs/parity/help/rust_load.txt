$ bpfman-rs load --help
Load an eBPF program on the system

Usage: bpfman-rs load <COMMAND>

Commands:
  file   Load an eBPF program from a local .o file
  image  Load an eBPF program packaged in a OCI container image from a given registry
  help   Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
