$ go-bpfman program --help
Usage: bpfman program <command> [flags]

Manage BPF programs.

Flags:
  -h, --help    Show help (-h for compact, --help for full).

Global Flags:
  --config=FILE              Config file path (default: /etc/bpfman/bpfman.toml)
                             ($BPFMAN_CONFIG).
  --log=SPEC                 Log spec (e.g., 'info,manager=debug')
                             ($BPFMAN_LOG).
  --runtime-dir=DIR          Root directory for runtime files
                             ($BPFMAN_RUNTIME_DIR).
  --image-cache-dir=DIR      Root directory for OCI image cache.
  --lock-timeout=DURATION    Timeout for acquiring the global writer lock (0 for
                             indefinite) ($BPFMAN_LOCK_TIMEOUT).

BPF Resources:
  program load file     Load from a local object file.
  program load image    Load from an OCI container image.
  program unload        Unload a managed BPF program.
  program get           Get details of a program by program ID.
  program list          List managed programs.
