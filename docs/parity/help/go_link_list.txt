$ go-bpfman link list --help
Usage: bpfman link list [flags]

List managed links.

Flags:
  -h, --help                     Show help (-h for compact, --help for full).

  -o, --output=text              Output format: text, json.
  -q, --quiet                    Output only link references (link/<id>),
                                 one per line.
      --program-id=PROGRAM-ID    Filter by program ID (supports hex with 0x
                                 prefix).
      --kind=KIND,...            Filter by link kind (e.g., --kind=xdp,kprobe).
  -p, --program-type=PROGRAM-TYPE,...
                                 Filter by the owning program's type (e.g.
                                 xdp,kprobe).
      --application=STRING       Filter by the owning program's application
                                 metadata.
  -m, --metadata-selector=METADATA-SELECTOR,...
                                 Filter by the owning program's KEY=VALUE
                                 metadata (can be repeated).

Global Flags:
  --config=FILE              Config file path (default: /etc/bpfman/bpfman.toml)
                             ($BPFMAN_CONFIG).
  --log=SPEC                 Log spec (e.g., 'info,manager=debug')
                             ($BPFMAN_LOG).
  --runtime-dir=DIR          Root directory for runtime files.
  --image-cache-dir=DIR      Root directory for OCI image cache.
  --lock-timeout=DURATION    Timeout for acquiring the global writer lock (0 for
                             indefinite) ($BPFMAN_LOCK_TIMEOUT).
