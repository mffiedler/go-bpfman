$ go-bpfman program list --help
Usage: bpfman program list [flags]

List managed programs.

Flags:
  -h, --help                  Show help (-h for compact, --help for full).

  -o, --output=text           Output format: text, json.
  -q, --quiet                 Output only program references (program/<id>),
                              one per line.
      --attached              Show only programs with active links.
      --unattached            Show only programs without active links.
      --type=TYPE,...         Filter by program type (case-insensitive, e.g.,
                              --type=xdp,kprobe).
  -p, --program-type=PROGRAM-TYPE,...
                              Filter by program type (Rust-compatible alias for
                              --type).
      --application=STRING    Filter by application metadata.
  -m, --metadata-selector=METADATA-SELECTOR,...
                              Filter by KEY=VALUE metadata (can be repeated).
  -a, --all                   Include unmanaged kernel programs (those loaded
                              outside bpfman).
  -l, --selector=STRING       Label selector (e.g., app=myapp,version!=v1).

Global Flags:
  --config=FILE              Config file path (default: /etc/bpfman/bpfman.toml)
                             ($BPFMAN_CONFIG).
  --log=SPEC                 Log spec (e.g., 'info,manager=debug')
                             ($BPFMAN_LOG).
  --runtime-dir=DIR          Root directory for runtime files.
  --image-cache-dir=DIR      Root directory for OCI image cache.
  --lock-timeout=DURATION    Timeout for acquiring the global writer lock (0 for
                             indefinite) ($BPFMAN_LOCK_TIMEOUT).
